# Boggle solver

## Usage

`python3.7 boggle_solver`

Example 5x5 grid:

`e i h v r in o y a m r l e u r s n s o c p e n i b`

Should also work with other python versions > 3.

## Options
- Change the dictionary by editing the value of DICT
